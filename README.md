# 《基于人工智能的定量社会科学互动学习指南》（中文版）（自我版）（草稿）
## 为什么要做这个?
在本人对于量化方法的学习实践过程中，发现有以下痛点：
1. 数学功底差，符号以及公式推导看不懂，课后题不会做，比较令人着急
2. 看懂了书本上的公式，却不知道其具体用处，理论和实际训练脱节，回馈周期慢，比学理论还沮丧
3. 程序反复报错，另开浏览器咨询AI，知识迭代速度慢

因此，本项目通过NotebookLLM将经典的方法著作拆解，并由此重新构建阅读笔记和训练材料生成Juypter文件，进而借助VScode或者Antigravity等与之类似的代码编辑器和边栏AI助手，得到精简读材料、快速回馈、迅速上手的方式进行“多快好省”的知识迭代。


## 你需要什么？
1. 一个类VScode编辑器
2. python环境配置
3. 一个代码Agent会员，如Codex、claude code或github coplilot
4. 一个脑子


## 项目里面有什么？
1. 可供直接阅读的笔记
2. 针对于笔记中的内容，使用边框中的AI助手进行通俗解释
3. Juypter笔记内代码编写，进行边看边做
4. 作业修改与订正
